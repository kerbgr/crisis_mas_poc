Supported File Formats
1. Nested Format (agent_profiles.json)
{
  "agents": [
    {
      "agent_id": "medical_expert_01",
      "name": "Dr. Dimitris Nikolaou",
      "role": "Medical Expert",
      "expertise": "emergency_medicine",
      "experience_years": 20,
      ...
    }
  ]
}
2. Flat Array Format (legacy experts.json)
[
  {
    "id": "expert_001",
    "name": "Dimitrios Papadopoulos",
    "role": "Senior Firefighter",
    "experience_years": 15,
    ...
  }
]

Both formats are automatically detected and supported by the web tools.
